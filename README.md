# Simple Interest Calculator

This project contains a Bash script (`simple-interest.sh`) that calculates simple interest based on user input.

## Description
The Simple Interest Calculator is a command-line tool built using Bash scripting. 
It prompts the user to enter:
- Principal amount
- Rate of interest
- Time period

The script then calculates the simple interest using the formula:

Simple Interest = (Principal × Rate × Time) / 100

## Usage
Run the script in a terminal:
```bash
bash simple-interest.sh
```
